Flexbox居中原理详解

为什么加上display:flex就可以实现居中？

Flexbox布局模型

Flexbox（弹性盒子布局）是一种一维布局模型，可以轻松控制项目在容器中的对齐、方向和顺序。

为什么需要display:flex？

display: flex 是启用Flexbox布局的关键。没有这个属性，Flexbox相关的对齐属性（如justify-content和align-items）将不会生效。

居中原理

当容器设置为 display: flex 后：

  • 容器内的子元素成为Flex项目
  • 可以使用 justify-content 控制项目在主轴上的对齐
  • 可以使用 align-items 控制项目在交叉轴上的对齐
  • 设置 justify-content: center 和 align-items: center 即可实现完美居中

在您的代码中的应用

在您的原始代码中，.content 类有 justify-content: center，但没有 display: flex，所以居中属性不会生效。

/* 您的原始代码（不生效） */ .content { justify-content: center; /* 这个属性需要display:flex才能工作 */ } /* 修正后的代码（生效） */ .content { display: flex; /* 启用Flexbox布局 */ justify-content: center; /* 现在这个属性可以工作了 */ align-items: center; /* 垂直居中 */ }
主轴 (justify-content)
交叉轴 (align-items)
居中元素
.container { display: flex; justify-content: center; align-items: center; }
